# 高中成绩统计系统

一个专为高中学生设计的成绩管理与分析工具，支持记录高一、高二、高三三年的所有考试成绩。

## 功能特点

- **成绩录入**：支持录入各次考试的成绩，包括月考、期中、期末等
- **成绩查看**：以卡片形式展示每次考试的详细成绩
- **统计分析**：自动计算各科平均分、最高分、最低分、得分率
- **趋势图表**：可视化展示成绩变化趋势，包含折线图、柱状图、雷达图
- **目标设定**：设置各科目标分数，追踪完成进度

## 支持科目

语文、数学、英语、物理、化学、生物、历史、地理、政治（共9科）

## 技术栈

- React 18
- Vite
- TailwindCSS
- Recharts（图表库）
- Lucide React（图标库）

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 数据存储

所有数据保存在浏览器的 localStorage 中，无需后端服务器。

## 使用说明

1. 点击"成绩录入"添加考试记录
2. 在"成绩查看"页面查看所有考试成绩
3. 在"统计分析"页面查看各科统计数据和能力分布图
4. 在"趋势图表"页面查看成绩变化趋势
5. 在"目标设定"页面设置学习目标并追踪进度
